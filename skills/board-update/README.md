# Board / Investor Update Agent

Builds structured board or investor updates with revenue performance, pipeline, team, and strategic narrative — everything vs. plan.

## Time saved

2-3 hours per update cycle, with a more consistent and professional format.

## Customization ideas

- Add your standard board deck template so the output matches your existing format
- Include your OKRs/KPIs so every update auto-references your goals
- Create a monthly version (lighter, email) and a quarterly version (full board deck)
- Connect to your finance tools via MCP for automated metric pulls
