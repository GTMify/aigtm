# Daily CRM Priorities — Cowork Prompt

Copy and paste this into Claude Cowork. Replace the `[bracketed fields]` with your details and paste your data inline.

---

```
Tell me what to work on today — the top 5-8 things that actually matter, not a 50-item task list.

## Today's calendar
[Paste meetings — time, company, contact, type]

## Open opportunities
[Paste opps as a table or list — name, amount, stage, close date, last activity, next step]

## Recent activity (last 7 days)
[Paste activity — what's happened recently with these accounts]

## Follow-ups due
[Paste any tasks, reminders, snoozed items]

## Context
Quota this quarter: $[X]
Current attainment: $[Y] ([Z]%)
Focus accounts: [Top 5-10 names]

## What I need
1. Top 5-8 priorities for today, ordered, with verb-led actions
2. Meeting prep for each of today's calls (1-line "know" + 1-line "drive")
3. Stalled deal alerts with suggested next actions
4. A short watchlist for the rest of the week
5. An honest "what I'm NOT doing today" list so I can override

## Rules
- Top 5-8 only. Cut ruthlessly.
- Every item is a verb + object, not just an account name.
- Show time estimates so I can block calendar.
- Be honest about what's missing from the data I pasted.
- Don't fabricate context I didn't give you.
```

---

**Note:** This skill works with pasted data. For live integrations (Salesforce, HubSpot, Gmail), install those tools separately and paste their output here.
