# Customer Risk / Churn Early Warning Agent

Assesses customer health across your portfolio, flags at-risk accounts by revenue impact, and prescribes specific save plays before renewal conversations.

## Time saved

1-2 hours per review cycle, but the real value is catching at-risk accounts 60-90 days earlier than you would have otherwise.

## Customization ideas

- Add your health score model so the agent uses your existing definitions
- Include your save playbook (executive alignment, re-onboarding, concession thresholds)
- Connect to your CS platform via MCP for automated health and usage data
- Create a "renewal prep" variant focused on accounts renewing in the next 90 days
