# Forecast Narrative Agent

Translates raw pipeline data into a polished forecast update for your manager, CRO, or board — with commit math, risk flags, and a gap analysis.

## Time saved

45-60 minutes per forecast cycle. More importantly, it forces you to name your commits with evidence, which improves forecast accuracy.

## Customization ideas

- Add your company's stage-to-probability mapping for weighted pipeline calculations
- Include historical win rates so the agent can reality-check your commit
- Connect to your CRM via MCP for automated data pulls
- Create a weekly version (lighter) and a monthly version (full detail)
