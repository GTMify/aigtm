# Post-Call Summary Agent

Turns raw call notes or transcripts into a structured internal summary, action items with owners, a follow-up email draft, and CRM-ready notes.

## Time saved

15-20 minutes per call, and ensures nothing falls through the cracks.

## Customization ideas

- Add your CRM's required fields so the CRM notes section matches your input format
- Include your company's follow-up email template for consistent formatting
- Connect to meeting recording tools (Granola, Gong, etc.) via MCP for automatic transcript ingestion
- Add deal qualification criteria (MEDDIC, BANT) so the agent scores the deal after each call
