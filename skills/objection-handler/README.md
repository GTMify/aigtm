# Objection Handler / Deal Unsticker

Diagnoses why a deal is stuck, prescribes a specific recovery play, and writes the exact messages to execute it — including a breakup option when walking away is the smartest move.

## Time saved

Less about time saved and more about deals saved. Gives sellers a structured approach instead of "I'll send another follow-up."

## Customization ideas

- Add your product's common objections with pre-built rebuttals
- Include case studies and ROI data so the agent can reference proof points
- Build a library of winning "breakup emails" from your team's best performers
- Add competitor-specific plays for "they chose [competitor]" scenarios
