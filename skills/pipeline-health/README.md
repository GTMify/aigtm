# Pipeline Health Check Agent

Audits your pipeline for stuck deals, coverage gaps, single-threaded risks, and produces a prioritized weekly action plan.

## Time saved

30-60 minutes per week compared to manual pipeline review and spreadsheet analysis.

## Customization ideas

- Add your team's specific stage probabilities instead of the defaults
- Customize risk thresholds (e.g., "stale after 14 days" instead of 30 for fast-cycle sales)
- Connect to your CRM via MCP for automatic data pulls
- Add multi-threading score based on contact count by seniority level
