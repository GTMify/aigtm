# Territory / Book of Business Analyzer

Analyzes a sales team's pipeline from above — rep performance, coverage gaps, whitespace, and account allocation recommendations.

## Time saved

1-2 hours per territory review, with more consistent and data-driven reallocation decisions.

## Customization ideas

- Add your team's segment definitions and account tiers for more precise analysis
- Include ramp timelines so new reps aren't unfairly flagged as underperforming
- Connect to your CRM via MCP for automated team-wide data pulls
- Create a "pre-QBR" variant that feeds into leadership pipeline reviews
