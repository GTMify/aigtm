# Inbox Triage — Cowork Prompt

Copy and paste this into Claude Cowork. Replace the `[bracketed fields]` with your details and paste your email content inline.

---

```
Triage my inbox. Categorize, prioritize, and draft replies.

## Emails to process
[Paste subject lines + senders, or full email bodies if you can]

Each email should include at minimum:
- From: [sender + relationship if you know it — customer / prospect / boss / vendor / personal]
- Subject: [subject line]
- Preview or body: [text]
- Date received (if known): [date]

## Context about me
My role: [What I do]
My voice in email: [Direct / friendly / formal — paste a sent message if you want me to match tone]

## What I need
1. Categorize each email (reply high urgency / reply low urgency / quick yes-no / FYI / action item / defer / unsub)
2. For Reply emails — draft a ready-to-send reply (with [brackets] for anything I need to fill in)
3. For Quick yes/no — a one-sentence answer
4. For Action items — pull them out as tasks with verb + object + due date
5. List senders or patterns I should consider unsubscribing from
6. Open questions where the right reply depends on info I haven't given

## Rules
- Drafts must be ready to send with one edit. Not "Thanks for reaching out."
- Match my voice based on the prior threads I've shared.
- Mark gaps explicitly with [brackets].
- Don't apologize unless warranted.
- The action item list goes to a task system — don't bury tasks in email replies.
- If you can't tell what to say, write the draft with a gap. Don't fabricate.
```

**Note:** This skill works with pasted email content. For live integration (Gmail / Outlook), install a dedicated tool separately and paste its output here.
