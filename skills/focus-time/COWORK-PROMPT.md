# Focus Time Planner — Cowork Prompt

Copy and paste this into Claude Cowork. Replace the `[bracketed fields]` with your details and paste your calendar and task list inline.

---

```
Plan my day around deep work. Find the focus windows, match tasks to slots, and protect my time.

## Today's calendar
[Paste meetings and existing blocks — time, title, length]
Example:
- 9:00 - 9:30  Daily standup
- 10:00 - 11:00 Customer call: Acme
- 12:30 - 13:00 Lunch (currently empty — should I protect this?)
- 14:00 - 15:30 Strategy meeting
- 16:00 - 16:30 1:1 with Sarah

## Tasks I want to do today
List with rough effort estimates:
- [Task] — [estimated time] — [type: deep work / shallow / communication / reactive / personal]
- [Task] — ...
- [Task] — ...

## About me
Morning person or night owl: [Morning / Night]
Typical deep-work block length I can sustain: [60 min / 90 min / 2 hr]
Things that always derail me: [Slack, manager pings, slack support channel]

## What I need
1. A time-blocked schedule for the day
2. Pre-block prep for each deep work block (files, tabs, 5-min starter)
3. Tasks getting deferred to tomorrow with why
4. Honest risks in the plan (meetings that run over, etc.)
5. Open questions if priorities aren't clear

## Rules
- Realistic, not optimistic. Account for context switching.
- Protect lunch.
- Don't stack deep work back-to-back.
- 25-min buffer after long meetings.
- If tasks don't fit, cut — don't compress.
- Match deep work to my energy curve.
```
