# Inbox Triage Agent

Categorizes, prioritizes, and drafts responses for your email backlog so you can hit inbox zero in minutes instead of hours.

## Time saved

20-30 minutes per day, plus the mental overhead of deciding what to reply to first.

## Customization ideas

- Connect to Gmail via MCP for automatic inbox scanning
- Add your VIP sender list so important contacts always get flagged 🔴
- Include your team roster so the agent knows who to suggest delegating to
- Create email response templates for recurring request types
