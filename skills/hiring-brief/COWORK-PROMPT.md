# Hiring Brief — Cowork Prompt

```
I need to hire a [role]. Help me scope the role, write the JD, and build the interview plan.

## The Role
Title: [e.g., Mid-Market Account Executive]
Level: [Junior / Mid / Senior / Lead / Director / VP]
Reports to: [Title]
Location: [City / Remote / Hybrid]
Why we're hiring: [Growth / Backfill / New market / Reorg]

## Context
Team size today: [N people]
What this person will sell / own: [Product, segment, deal size, cycle length]
Must-haves: [Non-negotiable skills or experience]
Nice-to-haves: [Bonus points]
Budget: $[range if known, or "need guidance"]

## What I need
1. **Job description** ready to post — lead with impact, not vague responsibilities. Outcomes like "Close $1.5M in ARR" not "Drive revenue growth."
2. **Comp guidance** with base, OTE, and equity ranges by percentile
3. **Interview scorecard** with:
   - Screening call questions (30 min)
   - Hiring manager deep-dive questions tied to competencies (45 min)
   - A skills exercise appropriate to the role
   - Panel interview assignments
   - Scoring rubric (1-4 scale with behavioral indicators)

## Rules
- No illegal or biased language in the JD.
- Don't over-credential. If it needs 3 years experience, don't write 7.
- Interview questions should be behavioral ("Tell me about a time..."), not hypothetical.
- Comp ranges are estimates — flag that I should verify against my own data.
```
