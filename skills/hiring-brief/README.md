# Hiring Brief / Role Scoper

Scopes a GTM role, writes a job description, provides comp guidance, and builds a structured interview plan with a scoring rubric.

## Time saved

3-4 hours per role, and produces more consistent, bias-resistant interview loops.

## Customization ideas

- Add your company's leveling framework so titles and comp bands align
- Include your culture values so the executive interview questions are tailored
- Add team-specific interview exercises (your actual mock discovery call scenario, etc.)
- Create variants for different role families (AE, SDR, CSM, RevOps)
