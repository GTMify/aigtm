# Deal Strategy Agent

Builds a strategic plan for a specific active deal — MEDDIC assessment, stakeholder map, competitive positioning, risk analysis, and a prioritized action plan.

## Time saved

30-45 minutes per deal review session, with better rigor because the framework is enforced consistently.

## Customization ideas

- Swap MEDDIC for your team's framework (BANT, Challenger, Force Management, SPICED)
- Add your product's specific competitive differentiators for sharper positioning
- Connect to your CRM via MCP to auto-pull deal history and contact data
- Include your pricing matrix so the agent can flag margin risks
