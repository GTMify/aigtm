# QBR / Executive Business Review Builder

Builds customer-facing quarterly business reviews that lead with value delivered, not product features — with adoption analysis, renewal context, and expansion framing.

## Time saved

2-3 hours per QBR. Also produces more consistent, customer-centric reviews across the team.

## Customization ideas

- Add your product's standard KPI benchmarks so the agent can compare customers
- Include your expansion playbook so growth opportunities are framed correctly
- Connect to your CS platform via MCP for automated health and usage data
- Create a "renewal QBR" variant that's specifically tuned for 90-day-out conversations
